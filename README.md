# Library Borrowing System

A Windows Forms application built with C# and .NET for managing library operations including book inventory, member registration, and borrowing transactions.

## Features
- Add and manage books in inventory
- Register and track library members
- Process book borrowing and returns
- Centralized management system

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- Object-Oriented Programming

## Project Overview
This project was developed as part of my App Development 1 module, demonstrating core software engineering principles and practical application development skills.
